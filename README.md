# Base do Projeto

A VMF Clothing surgiu com a necessidade de organizar melhor o controle de suas roupas, substituindo anotações manuais por um sistema simples e eficiente. Assim, foi desenvolvido um programa em linguagem C para facilitar o gerenciamento das peças da loja.

O sistema utiliza operações de CRUD para cadastrar, listar, atualizar e remover roupas, permitindo classificá-las por cor, tamanho e modelo (como camiseta, short e calça). Dessa forma, o controle do estoque se torna mais prático, rápido e organizado.
